DPT_GravityAlignment

Component Reference Guide
A procedural simulated pendulum utility designed to bypass rigid animation locking. It forces hanging objects (like counterweights or lanterns) to maintain gravity alignment while dynamically swaying in reaction to positional deltas using mathematical spring tension.

1. Inspector Property Configuration Tables

Gravity Aligned Elements Array

Processes an array of Pendulum blocks targeting hanging geometric structures within the prefab hierarchy.

Property Description
Label A descriptive text string used to organize and name array entries within the Unity Inspector layout for easy modification.
Target Transform The scene reference to the hanging object. The pivot point of this assigned mesh must sit exactly at the visual hinge connection point, not the center of mass.
Sway Sensitivity Determines how aggressively the geometric weight reacts to movement. You can input negative float values to invert the rocking direction if the object tilts incorrectly relative to its modeled orientation.
Gravity Tension The internal mathematical spring constant (Hooke's Law). Higher values pull the object back to a perfectly level state much faster, simulating a heavier, denser weight.
Damping The friction coefficient applied to the angular velocity. Higher values cause the rocking motion to settle rapidly, preventing infinite swinging loops.
Max Swing Angle X A hard clamp applied to the local X (Pitch) rotation angle. Limits how far forward or backward the object can swing before hitting physical resistance.
Max Swing Angle Z A hard clamp applied to the local Z (Roll) rotation angle. Limits how far side-to-side the object can sway. Often set lower than the X axis to simulate restricted hinge connections.
Disable Z Rotation A boolean toggle that completely disables all side-to-side (Roll) momentum. Useful for heavy, rigid axle connections where lateral swaying is physically impossible.

2. Technical Architecture

  • Procedural Simulated Physics: Eliminates the requirement for complex Unity Rigidbody component configurations, which often break commercial asset prefabs due to missing layer or physics matrix setup by end-users. All movement is faked procedurally.
  • Independent Execution Loop: Operates entirely decoupled from primary weapon tracking scripts. Evaluates exclusively within its own LateUpdate loop, measuring positional deltas after the Unity Animator has moved the parent structures.
  • Chassis Yaw Inheritance: While the X (Pitch) and Z (Roll) angles are mathematically overwritten to simulate gravity, the logic natively preserves the parent's Y (Yaw) rotation. This allows counterweights to swing freely while still turning left and right properly as the vehicle traverses.

3. Practical Assembly Configurations

Because the logic operates independently, it can be applied to any sub-mesh that requires reactive hanging physics without the overhead of the Unity physics engine.

Trebuchet Counterweight Setup Example:
  1. Attach the script to your trebuchet root prefab alongside your main mechanical drive component.
  2. Add an element block to the Gravity Aligned Elements array.
  3. Assign your hanging counterweight box to the Target Transform slot. Ensure the 3D model's pivot point is placed exactly at the top hinge rod, not the center of the wooden box.
  4. Set Sway Sensitivity to 50, Gravity Tension to 20, and Damping to 3 to simulate a massively heavy box of rocks that swings slightly when the arm drops but settles quickly into a resting state.

4. Troubleshooting Notes

"The counterweight is swinging in the opposite direction of the movement."
Invert the sign of your Sway Sensitivity property (e.g., change 50 to -50). Directional swing logic depends heavily on the default modeled orientation of your local transform axes.
"The counterweight starts violently shaking or spinning out of control."
This indicates a feedback loop caused by an incorrect pivot point. Ensure the assigned mesh has its origin point modeled precisely at the hinge connection point. If the pivot is at the center of mass, the script rotation will mathematically shift the hinge, creating a false movement delta every frame.
